
**Clause** is a law assistant aimed to help law practioners make quick and informed decisions. The idea is nto to replace the lawyers but to give them very fast retrieval of all the required info on a specific case. 

An example may include: the correct act, sub clauses, amendments, connected acts or clauses. 

Since Clause uses knowledge graphs extensively, it uses connected nature of various clauses and how can they be used for the benefit of the client in a given case.
